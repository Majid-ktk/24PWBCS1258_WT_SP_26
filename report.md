In this lab, I learned how to set up a professional development environment using 
Visual Studio Code and Git. I understood the purpose of version control and how Git 
helps in tracking every change made to a project over time. Creating and switching 
between branches helped me understand how developers work on new features without 
disturbing the stable main code. Merging branches showed me how all separate work 
eventually comes together into the final project. Overall, this lab gave me a strong 
practical foundation in Git and GitHub that I will continue to build on throughout 
this course.
